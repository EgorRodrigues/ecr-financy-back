Cadastro de Conta (Accounts)

Cria uma conta do tipo bank, wallet ou credit_card.

POST /accounts/ credit_card rules closing_day/due_day

Mapa visual

  1. Request autenticado Router recebe POST /accounts/ com JWT válido.
  2. Seleciona tenant get_db garante schema e ajusta search_path.
  3. Valida regras do schema Dias 1..31 e category_id apenas para credit_card.
  4. Persiste conta Cria accounts, faz commit e retorna AccountOut.

Resumo

Endpoint
POST /accounts/ (accounts.py)
Persistência
repositories/accounts.py:create_account
Tabela
accounts (models/accounts.py)
Isolamento por tenant
A sessão já está no schema do tenant via get_db.

Validações e regras

  • closing_day e due_day devem estar entre 1 e 31 (se informados).
  • category_id só é permitido quando type=credit_card (validação existe no schema e é reforçada no repository).
  • Se a regra de category_id falhar no repository, o router converte para 400 e faz rollback.

Referências: schemas/accounts.py:AccountCreate, repositories/accounts.py:create_account.

Diagrama (sequência)

sequenceDiagram
  autonumber
  actor Client as Cliente (Front/CLI)
  participant API as FastAPI Router (/accounts)
  participant Auth as verify_token/decode_token
  participant Dep as dependencies.get_db
  participant Val as Pydantic (AccountCreate)
  participant Repo as repositories.create_account

  Client->>API: POST /accounts/ (Bearer JWT) + payload
  API->>Auth: Depends(verify_token)
  Auth-->>API: payload (sub=UUID do usuário)
  API->>Dep: Depends(get_db)
  Dep->>DB: (SET search_path tenant_...)
  API->>Val: Valida payload (dias e regra category_id)
  alt Payload inválido (schema)
    Val-->>API: erro
    API-->>Client: 422 Unprocessable Entity
  else Payload válido
    Val-->>API: payload validado
    API->>Repo: create_account(session, payload)
    alt Regra inválida no repository
      Repo-->>API: ValueError
      API-->>Client: 400 Bad Request
    else OK
      Repo->>DB: INSERT accounts + COMMIT
      Repo-->>API: AccountOut
      API-->>Client: 200 OK + AccountOut
    end
  end
